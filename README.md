# PowerBI-data-Professional-survey-dashboard
Data Professional Survey Dashboard (Power BI)

# Project Overview

This project analyzes survey data from data professionals using Microsoft Power BI. The goal was to transform raw survey data into an interactive dashboard that highlights key trends, demographics, salaries, and job satisfaction across the data industry.

This project was completed as part of Alex The Analyst's Data Analyst Bootcamp.

# Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Modeling
- Power Query
- DAX
- Interactive Dashboards
- Data Visualization
- Business Intelligence

# Tools Used

- Microsoft Power BI

# Project Process

The following steps were completed:

- Imported the survey dataset into Power BI.
- Cleaned and transformed the data using Power Query.
- Created relationships and calculated measures where needed.
- Built interactive visualizations.
- Designed a professional dashboard to communicate key insights.

# Dashboard Insights

The dashboard provides insights into:

- Average salary by job title
- Salary by gender
- Country distribution
- Favorite programming languages
- Difficulty of breaking into data careers
- Job satisfaction and work-life balance
